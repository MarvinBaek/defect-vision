# MarkCloud Defect Detection Platform

AI 기반의 이미지 불량 인식 시스템의 MVP입니다.  
사용자는 이미지를 업로드하면, 서버에 연결된 AI 모델을 통해 부품 정상/불량 여부를 예측받을 수 있습니다.

---

## 기능 요약

- **Drag & Drop 이미지 업로드**
- **AI 모델을 통한 불량 예측 처리**
- **결과 상태 시각화 (정상 / 불량)**
